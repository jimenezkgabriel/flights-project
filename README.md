# Book Club Page 

## Project Overview
A responsive landing page for a book club platform where readers can discover new books, share recommendations, and join a community of book lovers. Features a cozy bookstore aesthetic with smooth interactions and user-friendly design.

## Features
Responsive navigation bar with book retailer links  
Search functionality for finding books  
Call-to-action section with smooth scroll  
Interactive signup form with validation  
Cozy bookstore aesthetic with warm color palette  
Smooth button animations and hover effects  
Mobile-responsive design  

## Technologies Used
- **HTML5** - Structure and semantic markup
- **CSS3** - Styling with gradients, animations, and flexbox
- **JavaScript (Vanilla)** - Form handling and smooth scroll interactions
- **Git & GitHub** - Version control and collaboration
- **Bootstrap** - CSS framework with helpful components such as the carousel component

## How to Run
1. Clone the repository:
   ```bash
   git clone [your-repo-url]
   ```
2. Navigate to the project folder:
   ```bash
   cd [folder-name]
   ```
3. Open `index.html` in your browser
4. Thats it lol. Enjoy the cozy book club experience!

## JavaScript Features
- **Smooth Scrolling**: CTA button smoothly scrolls to signup form
- **Form Validation**: Checks that all fields are filled before submission
- **Alert Confirmation**: Shows success message with user's name and email
- **Event Handling**: Manages form submission without page reload

## Challenges & Solutions
**Challenge**: Team member accidentally created duplicate folder  
**Solution**: Coordinated to copy code to correct repository and resolved branch divergence

**Challenge**: Choppy button hover animations  
**Solution**: Used CSS pseudo-elements with opacity transitions instead of background transitions for smooth GPU-accelerated animations

**Challenge**: Git merge conflicts and branch confusion  
**Solution**: Improved team communication, worked on different sections, and coordinated merge timing
